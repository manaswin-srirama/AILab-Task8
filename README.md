# Word Puzzle Game (Python Tkinter)

A simple Word Puzzle GUI game built using Python and Tkinter.

## Features
- Random word puzzle generation
- Score tracking
- Countdown timer
- Hint letters
- Dark/Light theme toggle
- New puzzle generation

## Technologies Used
- Python
- Tkinter
- Random module

## How to Run

1. Clone the repository

git clone https://github.com/yourusername/word-puzzle-game.git

2. Navigate into the folder

cd word-puzzle-game

3. Run the program

python task8.py

## Game Rules
- A hidden word is placed horizontally in the grid
- Some letters are revealed as hints
- Fill in the remaining letters
- Press **Check** to verify
- Correct answer gives **10 points**
